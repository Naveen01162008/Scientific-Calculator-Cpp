# Scientific-Calculator-Cpp
Description 
A menu-driven scientific calculator built using Object-Oriented programming in C++.
Features
- Basic operations (Add,Subtract,Multiply,Divide)
- Scientific Functions (Sin,Cos,Log,Sqrt,Exponent)
- User-friendly menu interface
Concepts Used
- Classes and Objects
- Functions
- Loops and Conditionals
- Standard Library ('<cmath>')
How to run
- g++ Scientific_Calculator.cpp -o calc
- followed by ./calc
